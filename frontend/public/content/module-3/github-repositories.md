# GitHub Repositories

GitHub is a cloud platform that hosts Git repositories. It layers a powerful social and collaborative interface on top of Git's version control.

---

### Exploring a GitHub Repository

When visiting a project repository on GitHub, you'll see several key tabs:

- **Code**: The file explorer showing branch directories, README, and licenses.
- **Issues**: Task boards where bugs are reported and feature requests are tracked.
- **Pull Requests**: Code proposals waiting for peer reviews and merging.
- **Discussions**: The community forum for questions, ideas, and general chats.
- **Actions**: The automated workflow engine that builds code and runs tests (CI/CD).

### Star and Watch

- **Star ⭐**: Similar to bookmarking. It shows appreciation and registers your interest.
- **Watch 👀**: Subscribes you to notifications (issues, PRs, releases) so you can follow the project's development live.
