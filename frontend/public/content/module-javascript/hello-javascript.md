# Hello, JavaScript

Welcome to the JavaScript/TypeScript Execution Sandbox! 

JavaScript is the programming language of the web. In this lesson, we will learn how to write basic JavaScript functions and see the output directly in the browser using Web Workers.

## What is a Web Worker?

A Web Worker allows you to run JavaScript in the background, completely separate from the main browser thread. This means even if your code takes a long time to run (or accidentally enters an infinite loop), it won't freeze the webpage!
